---
title: Scrape
description: "POST /v1/scrape тАФ single URL to content, synchronously."
---

Fetch a single URL and return its content inline.

```bash
curl https://api.qrawl.dev/v1/scrape \
  -H "Authorization: Bearer qr-..." \
  -H "Content-Type: application/json" \
  -d '{ "url": "https://example.com", "format": "markdown" }'
```

## Body

| Field | Type | Default | Notes |
| --- | --- | --- | --- |
| `url` | string | тАФ | Required. The page to scrape. |
| `format` | `markdown` \| `json` \| `html` \| `text` | `markdown` | Output format. |
| `screenshot` | boolean | `false` | Capture a screenshot (Pro). |
| `jsRendering` | boolean | `false` | Render with a headless browser (Pro). |
| `piiFilter` | boolean | `false` | Redact PII from output (Pro). |

## Response

```json
{
  "success": true,
  "data": {
    "url": "https://example.com",
    "content": "# Example Domain\n\n...",
    "metadata": {
      "title": "Example Domain",
      "description": "...",
      "wordCount": 312,
      "links": ["https://www.iana.org/domains/example"]
    }
  },
  "requestId": "req_..."
}
```
